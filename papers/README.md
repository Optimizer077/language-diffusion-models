# Papers referenced in this course

The PDFs in this folder are the papers each lesson is built on — downloaded from
[arXiv](https://arxiv.org/) for convenient offline study. Each is linked to its
canonical arXiv page below.

> **Copyright / redistribution note.** These PDFs are the authors' works, hosted on
> arXiv under arXiv's licensing. arXiv lets you *download* them freely, but
> *redistributing* PDFs in a public repository is a gray area (licenses vary per
> paper). If you publish this repo and would rather not redistribute the PDFs, add
> `papers/*.pdf` to `.gitignore` — the links below give readers the same access.
> The scholarship belongs to the authors; this course only teaches from it.
